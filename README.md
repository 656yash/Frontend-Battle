# Frontend Battle

This repository contains the frontend build for **AxiomAI | Precision AI Automation Platform**. It was built as part of a frontend hackathon/battle.

## Overview

AxiomAI is a single-page landing site that showcases:
- A custom WebGL shader background
- A live execution pipeline animation
- An interactive Bento-to-Accordion feature showcase section
- Reactive pricing logic

## Live Demo

The site is configured to be deployed on Vercel or any standard static hosting platform using the root `index.html` file.
